# Speech Repository

This repository contains tools and benchmarks for experimenting with Google Cloud Speech APIs, specifically focusing on Automatic Speech Recognition (ASR) and Text-to-Speech (TTS) capabilities using the modern `google-genai` SDK.

## Project Structure

*   **`STT/`**: Contains core Speech-to-Text provider implementations.
    *   `stt_providers/`: Strategy pattern implementations for different STT engines (e.g., Gemini).
    *   `vad.py`: Voice Activity Detection utilities.
    *   `examples/`: Example scripts for short and long audio processing.
*   **`benchmark/`**: Pipeline for evaluating STT performance.
    *   `run_benchmark.py`: Main script to orchestrate the benchmark.
*   **`assets/`**: Data assets used and produced by the system.
    *   `batch/`: Stored pairs of audio (`.wav`) and text (`.txt`) files used for batch processing.
    *   `benchmark_output/`: Results generated by the benchmark pipeline.
*   **`music/`**: Subdirectory grouping Lyria-related journal projects.

## Getting Started

1.  **Environment Setup**:
    Ensure you have the required dependencies installed in your virtual environment.
    ```bash
    source .venv/bin/activate
    pip install webrtcvad pydub google-genai "setuptools<81"
    ```
2.  **Running the Benchmark**:
    Use the provided shell script to run the benchmark pipeline:
    ```bash
    ./run_benchmark.sh
    ```

## Key Features

*   **Modern SDK**: Uses `google-genai` for interacting with Gemini models.
*   **Strategy Pattern**: Easy to plug in new STT providers by implementing `BaseSTTProvider`.
*   **Optimized VAD**: Supports in-memory byte processing to reduce disk I/O.
